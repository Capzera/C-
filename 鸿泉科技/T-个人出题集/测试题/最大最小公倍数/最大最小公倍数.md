# 最大最小公倍数

## 题目背景



## 题目描述

​		已经一个正整数$N$，问从$1 - N$中任选出**三个数**，他们最小公倍数的最大值可以为多少。

## 输入格式

​		一个正整数$N$

## 输出格式

​		一个整数表示答案

## 样例 #1

### 样例输入 #1

```
3
```

### 样例输出 #1

```
6
```

## 样例 #2

### 样例输入 #2

```
5
```

### 样例输出 #2

```
60
```

## 提示

对于$30\%$的数据，有$1 \leq N \leq 10 ^ 4$。

对于$100\%$的数据，有$1 \leq N \leq 10^6$。
