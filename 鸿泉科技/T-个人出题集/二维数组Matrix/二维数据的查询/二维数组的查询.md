# 二维数组的查询

## 题目背景

## 题目描述

输入两个正整数$n,m$，表示数组$a$的行列，请输出第$x$行$y$列的值。

## 输入格式

第一行两个正整数$n, m$，表示数组$a$的行列。

接下来$n$行$m$列，表示数组的值。

接下来一行有$2$个整数$x, y$。表示需要查询第$x$行$y$列的值。

## 输出格式

一个整数，表示答案。

## 样例 #1

### 样例输入 #1

```
3 3
1 4 7
2 5 8
3 6 9
1 1
```

### 样例输出 #1

```
5
```

## 提示

对于$100\%$的数据，有$1 \leq n, m, x, y \leq 3000, 1 \leq a_i,x \leq 10^6$。
