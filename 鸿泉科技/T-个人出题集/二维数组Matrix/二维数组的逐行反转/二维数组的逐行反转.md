# 二维数组的逐行反转

## 题目背景

## 题目描述

给定一个$n × m$的二维数组，将该数组的每行元素反转后输出。

## 输入格式

第一行两个正整数$n, m$，表示数组的行列。

接下来$n$行$m$列，表示数值的值。

## 输出格式

$n$行$m$列正整数，表示数组逐行反转后的值。

## 样例 #1

### 样例输入 #1

```
3 3
1 2 3
4 5 6
7 8 9
```

### 样例输出 #1

```
3 2 1
6 5 4
9 8 7
```

## 提示

对于$100\%$的数据，有$1 \leq n, m \leq 10^3$ , $1 \leq a_{ij} \leq 10^6$。
