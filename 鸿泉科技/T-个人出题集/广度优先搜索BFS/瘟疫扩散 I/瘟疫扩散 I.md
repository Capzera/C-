# 瘟疫扩散 I

## 题目背景

## 题目描述

给你一个$n × m$的矩阵，并输入一个瘟疫的坐标$(x, y)$，这个瘟疫**每秒**会扩散`上下左右`四格，请你输出一个$n × m$的矩阵，每一格表示瘟疫最短需要多少秒会扩散到这一格。

## 输入格式

四个整数，$n，m，x，y$分别代表矩阵的长宽及瘟疫的坐标。

## 输出格式

一个$n × m$的矩阵，代表答案。

## 样例 #1

### 样例输入 #1

```
3 3 1 1
```

### 样例输出 #1

```
2 1 2
1 0 1
2 1 2
```

## 提示

对于$100\%$的数据，有 $1 \leq n, m \leq 1000$。
