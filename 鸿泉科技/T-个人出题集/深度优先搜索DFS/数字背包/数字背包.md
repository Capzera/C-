# 数字背包

## 题目背景

## 题目描述

你现在拥有数字卡片`[1,2,3,4,5,6,7,8,9]`各一张，你现在需要拿出其中的$k$张放进背包中，这个背包会对这些卡片求和。

请你输出所有和为$n$的方案。

## 输入格式

第一行两个正整数$n, k$。

## 输出格式

若干行，每行表示一种方案，输出按照字典序排列，并且每种方案按照数字卡片从小到大排列。

## 样例 #1

### 样例输入 #1

```
7 3
```

### 样例输出 #1

```
1 2 4
```

## 提示

对于$100\%$的数据，有$1 \leq n \leq 45, 1 \leq k \leq 9$。
