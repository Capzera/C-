# 岛屿数量

## 题目背景

## 题目描述

给定一个$[0, 1]$矩阵，其中`0`代表水域，`1`代表陆地，求这块地岛屿的数量。

注意：一个陆地四个方向上如果仍然有陆地，则表示这块陆地是连续的区域。

此外，你可以假设该网格的四条边均被水域包围。

## 输入格式

第一行两个正整数$n, m$，表示这块地区的面积。

接下来$n$行$m$列，表示这块地区的水域陆地情况。

## 输出格式

一个整数，表示岛屿的数量。

## 样例 #1

### 样例输入 #1

```
3 3
1 0 0
0 1 1
0 1 1
```

### 样例输出 #1

```
2
```

## 提示

对于$100\%$的数据，有$1 \leq n, m \leq 300$。$a[i][j]$的值为`0`或`1`。
