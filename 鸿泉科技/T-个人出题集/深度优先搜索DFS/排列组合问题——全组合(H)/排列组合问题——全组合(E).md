# 排列组合问题——全组合(H)

## 题目背景



## 题目描述

输入两个正整数$n， k$，输入$n$个可能含有重复元素的数组，请输出这个数组中$k$个元素的组合情况。

## 输入格式

第一行，两个正整数$n, k$。

第二行，$n$个整数，表示数组的值。

## 输出格式

若干行，一行输出一个排列，用空格隔开。

**请按照字典序输出！！**

## 样例 #1

### 样例输入 #1

```
4 2
1 1 2 2
```

### 样例输出 #1

```
1 1
1 2
2 2
```

## 提示

对于$100\%$的数据，有$1 \leq n, k \leq 30, 1 \leq a_i \leq n$。
