# 动态和

## 题目背景

## 题目描述

给定一个数组`a`，请输出该数组的动态和的数组。

对于数组第$i$位的动态和表示为$a[0] + a[1] + ... + a[i - 1]$。

## 输入格式

第一行一个正整数$n$，表示数组的长度

第二行$n$个整数，表示题目中的数组。

## 输出格式

$n$个整数，表示数组的动态和数组

## 样例 #1

### 样例输入 #1

```
5
1 2 3 4 5
```

### 样例输出 #1

```
1 3 6 10 15
```

## 提示

对于$30\%$的数据，有$1 \leq n \leq 1000$， $0 \leq a[i] \leq 1000$

对于$100\%$的数据，有$1 \leq n \leq 10^5$， $-1000 \leq a[i] \leq 1000$
