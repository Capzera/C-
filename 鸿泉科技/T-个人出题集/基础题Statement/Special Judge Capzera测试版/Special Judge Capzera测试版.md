# Special Judge Capzera测试版

## 题目描述

输入一个数$n$，若$n$为奇数则输出"Odd"，否则输出"Even"。

## 输入格式

一个整数$n$。

## 输出格式

一个字符串，输出`Odd`， `ODD`， `odd`，`Even`，`EVEN`，`even`均可。

## 样例 #1

### 样例输入 #1

```
2
```

### 样例输出 #1

```
Even
```

## 样例 #2

### 样例输入 #2

```
3
```

### 样例输出 #2

```
Odd
```

## 提示

对于$100\%$的数据，有$1 \leq n \leq 100$。
