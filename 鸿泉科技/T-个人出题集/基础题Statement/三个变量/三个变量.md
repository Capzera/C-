# 三个变量

## 题目背景

## 题目描述

给定三个变量`a, b, c`，现在需要将三个变量交换后输出。

交换规则如下，每个变量保存上一个变量的值，变量`a`保存变量`c`的值。

注意：题目要求真正的做到修改`a, b, c`三个变量，不要直接在输出中修改三个变量的顺序。

如：

```cpp
int a, b, c;
cout << "a = " << c << ", b = " << a << ", c = " << b << endl;
```

这样写是不合规范的。

## 输入格式

一行三个整数`a, b, c`，用空格隔开。

## 输出格式

交换后`a, b, c`的值，注意输出的格式。

## 样例 #1

### 样例输入 #1

```
2 5 3
```

### 样例输出 #1

```
a = 3, b = 2, c = 5
```
