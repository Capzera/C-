# 斐波那契数列Plus

## 题目描述

有这样的一个数列：`[1 1 1 3 5 9 17....]`

输入一个整数$n$，求解这个数列的第$n$项。

## 输入格式

一个整数$n$。

## 输出格式

一个整数，表示答案。

## 样例 #1

### 样例输入 #1

```
4
```

### 样例输出 #1

```
3
```

## 样例 #2

### 样例输入 #2

```
6
```

### 样例输出 #2

```
9
```

## 提示

对于$30\%$的数据，有$1 \le n \le 10$。

对于$100\%$的数据，有$10 \le n \le 40$。
