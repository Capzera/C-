# 判断因数

## 题目描述

输入两个整数`n`，`m`，如果`m`是`n`的因数，则输出`Yes`，否则输出`No`。

如果一个数`a`是另一个数`b`的因数，则认为`b`除以`a`没有余数刚好整除，`a`一定不是`0`。

例如：`3`是`18`的因数，因为$18 ÷3$没有余数。

## 输入格式

第一行两个整数`a`，`b`。

## 输出格式

一个字符串，`Yes`或`No`。

## 样例输入 #1

```
6 2
```

## 样例输出 #1

```
Yes
```

## 输入样例 #2

```
7 5
```

## 样例输出 #2

```
No
```
