# 三角形判断

## 题目描述

输入三角形的三条边，如果他们能组成一个三角形，则输出`It is a triangle`，如果不能，则输出`It isn't a triangle`。

## 输入格式

一行三个整数$a, b, c$。表示三角形的三条边

## 输出格式

一行，一个字符串，表示结果。

## 样例 #1

### 样例输入 #1

```
6 6 6
```

### 样例输出 #1

```
It is a triangle
```

## 样例 #2

### 样例输入 #2

```
7 1 1
```

### 样例输出 #2

```
It isn't a triangle
```

## 提示

对于$100\%$的数据，有$1 \leq a, b, c \leq 10^4$
