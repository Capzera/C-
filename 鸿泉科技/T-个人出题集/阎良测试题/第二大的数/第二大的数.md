# 第二大的数

## 题目背景

## 题目描述

给定一个数组，请输出这个数组中的第二大的数。

如果第二大的数字不存在，请输出$No$

注意：第二大的数指的是排名第二，而不是排序后位于第二个位置的数。

## 输入格式

第一行一个正整数$n$，表示数组的长度

第二行$n$个正整数，表示题目中的数组。

## 输出格式

一个整数表示第二大的数，或$No$

## 样例 #1

### 样例输入 #1

```
6
1 3 2 2 3 2
```

### 样例输出 #1

```
2
```

## 提示

对于$100\%$的数据，有$1 \leq n \leq 10^6$， $ -10^9 \leq a[i] \leq 10^9$
