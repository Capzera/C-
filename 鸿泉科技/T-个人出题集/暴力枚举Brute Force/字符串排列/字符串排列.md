# 字符串排列

## 题目背景

## 题目描述

给定一个字符串`s`，求该字符串从小到大按照字典序的排列。

例如字符串`acb`的排列为：

```
abc
acb
bac
bca
cab
cba
```



## 输入格式

一行，一个字符串`s`。

## 输出格式

若干行，`S`的全排列。

## 样例 #1

### 样例输入 #1

```
cab
```

### 样例输出 #1

```
abc
acb
bac
bca
cab
cba
```

## 样例 #2

### 样例输入 #2

```
bab
```



### 样例输出 #2

```
abb
bab
bba
```



## 提示

对于$100\%$的数据，有$1 \leq s.size() \leq 9$ 。
